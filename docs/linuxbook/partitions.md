# Filesystems 

- ext2
- ext3: journaling function (save metadata im journaling system)
- ext4: RAM Cash
- xfs: Journaling and high speed
- vfat: Windows Filesystem
- exFat: Flash Memory
- btrfs: big memory, copy on write, Railsystem, btrfs-convert tool


## fdisk 
Program to do partitioning

List partitions:

    fdisk -l
